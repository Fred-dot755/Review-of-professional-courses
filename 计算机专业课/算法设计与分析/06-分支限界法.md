## 6.1 分支限界法基本思想

### 6.1.1 什么是分支限界法

**分支限界法**（Branch and Bound）是一种在解空间树上搜索问题最优解的算法。与回溯法不同，分支限界法采用**广度优先**或**最小代价优先**的策略搜索解空间树。

```
分支限界法 = 广度优先/最佳优先搜索 + 剪枝（限界）
```

### 6.1.2 分支限界法 vs 回溯法

| 维度 | 回溯法 | 分支限界法 |
|------|--------|------------|
| 搜索策略 | 深度优先（DFS） | 广度优先（BFS）或最佳优先 |
| 目标 | 找出**所有解** | 找出**最优解** |
| 数据结构 | 系统栈（递归） | 队列/优先队列 |
| 节点存储 | 只存当前路径 | 每个活节点都需存完整状态 |
| 空间复杂度 | O(解空间深度) | O(活节点数) |
| 剪枝效率 | 依赖剪枝函数 | 限界函数可更早剪枝 |

### 6.1.3 两种搜索方式

#### （1）队列式分支限界法（FIFO）

活节点表组织为**普通队列**，按先进先出原则选取下一个扩展节点。

#### （2）优先队列式分支限界法（Priority Queue）

活节点表组织为**优先队列**（通常是最小堆或最大堆），按节点的优先级（通常与目标函数相关）选取最有希望的节点扩展。

### 6.1.4 分支限界法的一般框架

```c
void BranchBound() {
    // 初始化活节点队列
    Queue q = CreateQueue();
    
    // 根节点入队
    Enqueue(q, root);
    
    while (!IsEmpty(q)) {
        Node *current = Dequeue(q);  // FIFO或取优先级最高
        
        if (IsSolution(current) && current->value > bestv) {
            bestv = current->value;  // 更新最优解
        }
        
        // 扩展子节点
        for each 子节点 child of current {
            if (满足约束条件) {
                计算 child 的限界值 bound
                if (bound > bestv)  // 可能有更优解
                    Enqueue(q, child);
            }
        }
    }
}
```

## 6.2 0-1 背包问题（分支限界法）

### 6.2.1 优先队列式分支限界

每个活节点包含：当前重量 cw、当前价值 cv、上界 bound、当前层数 level。

**上界计算**：同回溯法，用分数背包贪心策略估计剩余物品的最大可能价值。

**优先队列**：按 bound 值从大到小排序（最大堆），优先扩展 bound 最大的节点。

### 6.2.2 算法实现

```c
typedef struct Node {
    int level;       // 当前决策到第几个物品
    int cw, cv;      // 当前重量, 当前价值
    double bound;    // 上界
} Node;

// 计算上界
double Bound(Node node, int n, int C, int w[], int v[]) {
    if (node.cw >= C) return 0;
    
    double cleft = C - node.cw;
    double bound = node.cv;
    int i = node.level + 1;
    
    while (i <= n && w[i] <= cleft) {
        cleft -= w[i];
        bound += v[i];
        i++;
    }
    if (i <= n)
        bound += v[i] * cleft / w[i];
    return bound;
}

int KnapsackBB(int n, int C, int w[], int v[]) {
    // 按 v/w 降序排序物品（略）
    
    PriorityQueue *pq = CreateMaxHeap();
    int bestv = 0;
    
    // 创建根节点
    Node root = {0, 0, 0, 0};
    root.bound = Bound(root, n, C, w, v);
    Insert(pq, root);
    
    while (!IsEmpty(pq)) {
        Node current = ExtractMax(pq);
        
        if (current.bound > bestv) {
            // 左子节点：装当前物品
            Node left = current;
            left.level = current.level + 1;
            left.cw += w[left.level];
            left.cv += v[left.level];
            
            if (left.cw <= C && left.cv > bestv)
                bestv = left.cv;
            
            left.bound = Bound(left, n, C, w, v);
            if (left.bound > bestv)
                Insert(pq, left);
            
            // 右子节点：不装当前物品
            Node right = current;
            right.level = current.level + 1;
            right.bound = Bound(right, n, C, w, v);
            if (right.bound > bestv)
                Insert(pq, right);
        }
    }
    return bestv;
}
```

## 6.3 旅行商问题（TSP）— 分支限界法

### 6.3.1 优先队列式分支限界

每个活节点代表一条部分路径。**优先级** = 当前路径长度的下界估计值（越小越优先）。

### 6.3.2 下界（限界函数）设计

一个经典的下界：**行规约 + 列规约后的矩阵的归约常数和**。

```
下界 = 当前路径长度 + 归约后的矩阵的归约常数和
```

归约：每行减去该行最小元素，每列减去该列最小元素。所有减去的值之和即为当前路径的下界估计。

### 6.3.3 算法概要

```c
typedef struct Node {
    double cost;       // 当前路径代价
    double lb;         // 下界
    int level;         // 已访问城市数
    int path[N];       // 当前路径
    int visited[N];    // 访问标记
} Node;

double TSP_BranchBound(int graph[][N], int n) {
    PriorityQueue *pq = CreateMinHeap();  // 按lb最小优先
    double bestc = INF;
    
    Node root;
    root.cost = 0;
    root.level = 1;
    root.path[0] = 1;  // 从城市1出发
    root.lb = ComputeLB(graph, &root, n);
    Insert(pq, root);
    
    while (!IsEmpty(pq)) {
        Node current = ExtractMin(pq);
        
        if (current.lb >= bestc) continue;  // 不可能更优
        
        if (current.level == n) {
            // 所有城市已访问，构成回路
            double total = current.cost + graph[current.path[n-1]][1];
            if (total < bestc) {
                bestc = total;
                // 记录最优路径
            }
        } else {
            int lastCity = current.path[current.level - 1];
            for (int i = 2; i <= n; i++) {
                if (!current.visited[i] && graph[lastCity][i] != INF) {
                    Node child = current;
                    child.path[child.level] = i;
                    child.visited[i] = 1;
                    child.cost += graph[lastCity][i];
                    child.level++;
                    child.lb = ComputeLB(graph, &child, n);
                    
                    if (child.lb < bestc)
                        Insert(pq, child);
                }
            }
        }
    }
    return bestc;
}
```

### 6.3.4 复杂度分析

- 最坏情况：O(n!)，但实际通过限界剪枝大幅减少
- 分支限界法通常比回溯法更早找到最优解

## 6.4 装载问题

### 6.4.1 问题描述

有 n 个集装箱，重量分别为 wᵢ。有两艘船，载重分别为 C₁ 和 C₂，且 Σwᵢ ≤ C₁+C₂。问是否存在一种装载方案，将所有集装箱装上两艘船。

等效于：将集装箱分成两个子集，一个子集的总重 ≤ C₁，另一个的总重 ≤ C₂。

由于 Σwᵢ ≤ C₁+C₂，只需找到总重 ≤ C₁ 且**最大**的子集。

### 6.4.2 队列式分支限界

```c
typedef struct QNode {
    int level;    // 当前处理第几件
    int cw;       // 当前装载重量
} QNode;

int MaxLoading(int w[], int C, int n) {
    Queue *q = CreateQueue();
    int bestw = 0;
    int r = 0;  // 剩余集装箱总重
    for (int i = 2; i <= n; i++) r += w[i];
    
    QNode root = {1, 0};
    Enqueue(q, root);
    
    // 添加层结束标记 -1
    Enqueue(q, (QNode){-1, -1});
    int i = 1;
    
    while (!IsEmpty(q)) {
        QNode current = Dequeue(q);
        
        if (current.level == -1) {  // 层结束标记
            if (IsEmpty(q)) break;
            Enqueue(q, (QNode){-1, -1});
            i++;
            r -= w[i];
            continue;
        }
        
        int wt = current.cw + w[i];
        
        // 左子节点（装入）
        if (wt <= C) {
            if (wt > bestw) bestw = wt;
            if (i < n) Enqueue(q, (QNode){i+1, wt});
        }
        
        // 右子节点（不装）+ 限界
        if (current.cw + r > bestw)  // 还有可能超过当前最优
            if (i < n) Enqueue(q, (QNode){i+1, current.cw});
    }
    return bestw;
}
```

### 6.4.3 优先队列式优化

改用最大堆，按 **当前重量 + 剩余总重**（即可能达到的最大重量）作为优先级：

```c
typedef struct PQNode {
    int level, cw;
    int ub;  // 上界 = cw + r
} PQNode;

// 优先级 = ub，最大堆
```

这样优先扩展最有希望达到最大重量的节点，能更快找到最优解。

### 习题2

> 分支限界法中，优先队列式和队列式的主要区别是什么？

**解**：

| | 队列式(FIFO) | 优先队列式 |
|---|---|---|
| 选取策略 | 先进先出 | 按优先级（最优优先） |
| 找到最优解速度 | 较慢 | 通常更快 |
| 数据结构 | 普通队列 | 堆（优先队列） |
| 典型应用 | 装载问题 | 0-1背包、TSP |

优先队列式通常能更快找到最优解，因为总是从最有希望的节点扩展，有利于提高限界值从而剪枝。

### 习题3

> TSP 问题用分支限界法求解时，如何设计好的限界函数？

**解**：好的下界函数包括：

1. **行列归约法**：对剩余距离矩阵做行归约和列归约，归约常数之和为下界
2. **最小出边法**：每个未访问城市取最短出边，求和
3. **1-tree 下界**：构造一个包含所有顶点、且某一指定顶点度数为2的生成树，求其最小权值

实际中常用**行列归约法**，计算简单且下界较紧。
