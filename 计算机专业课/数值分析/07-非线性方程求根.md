# 数值分析·第7篇：非线性方程求根

## 一、问题描述

求 $f(x) = 0$ 的根 $x^*$，其中 $f(x)$ 为非线性函数。

例如：
- $x^3 - x - 1 = 0$
- $e^{-x} - \sin x = 0$

> 非线性方程通常无解析解，需用迭代法求数值解。

---

## 二、二分法（Bisection）

### 2.1 前提条件

$f \in C[a,b]$ 且 $f(a) \cdot f(b) < 0$（介值定理保证存在根）。

### 2.2 算法

```
while (b - a) / 2 > ε:
    c = (a + b) / 2
    if f(c) == 0: break
    if f(a) * f(c) < 0:
        b = c
    else:
        a = c
root = (a + b) / 2
```

### 2.3 误差估计

经过 $k$ 步后：

$$
|x_k - x^*| \le \frac{b-a}{2^{k+1}}
$$

**收敛阶**：线性收敛（每步误差减半）。

| 优点 | 缺点 |
|------|------|
| 必定收敛 | 收敛慢 |
| 简单可靠 | 需已知含根区间 |
| | 不能求复根和重根 |

---

## 三、不动点迭代

### 3.1 不动点形式

将 $f(x) = 0$ 改写为：

$$
x = \varphi(x)
$$

若 $x^* = \varphi(x^*)$，则 $x^*$ 是 $\varphi$ 的不动点。

### 3.2 迭代格式

$$
x_{k+1} = \varphi(x_k)
$$

### 3.3 收敛性定理

若 $\varphi$ 在 $[a,b]$ 上满足：
1. $\varphi(x) \in [a,b]$（自映射）
2. $|\varphi'(x)| \le L < 1$（压缩映射）

则 $\varphi$ 在 $[a,b]$ 上存在唯一不动点，且迭代收敛。

**收敛阶**：

若 $\varphi'(x^*) \ne 0$，则**线性收敛**；若 $\varphi'(x^*) = 0$，则可能超线性收敛。

---

## 四、Newton 迭代法

### 4.1 几何意义

用切线逼近曲线，切线与 $x$ 轴的交点作为新近似值。

### 4.2 迭代格式

$$
x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)}
$$

对应不动点函数：$\varphi(x) = x - \frac{f(x)}{f'(x)}$

### 4.3 收敛性

**单根**：至少二阶收敛（$\varphi'(x^*) = 0$）。

**重根**：若 $x^*$ 是 $m$ 重根（$m > 1$），则退化为线性收敛。可改用：

$$
x_{k+1} = x_k - m\frac{f(x_k)}{f'(x_k)}
$$

恢复二阶收敛。

### 4.4 收敛准则

| 准则 | 条件 |
|------|------|
| 残差 | $|f(x_k)| < \varepsilon_1$ |
| 步长 | $|x_{k+1} - x_k| < \varepsilon_2$ |

### 4.5 收敛阶证明

$$
\begin{aligned}
\varphi(x) &= x - \frac{f(x)}{f'(x)} \\
\varphi'(x) &= \frac{f(x)f''(x)}{[f'(x)]^2} \\
\varphi'(x^*) &= 0 \quad (\because f(x^*) = 0,\ f'(x^*) \ne 0)
\end{aligned}
$$

$$
\varphi''(x^*) = \frac{f''(x^*)}{f'(x^*)} \ne 0
$$

故单根时 Newton 法为二阶收敛。

### 优缺点

| 优点 | 缺点 |
|------|------|
| 收敛快（二阶） | 需要导数 $f'(x)$ |
| 单根附近高效 | 对初值敏感 |
| | 每步导数计算可能昂贵 |

---

## 五、割线法（Secant Method）

### 5.1 基本思想

用差商 $\frac{f(x_k) - f(x_{k-1})}{x_k - x_{k-1}}$ 代替 $f'(x_k)$。

### 5.2 迭代格式

$$
x_{k+1} = x_k - f(x_k) \cdot \frac{x_k - x_{k-1}}{f(x_k) - f(x_{k-1})}
$$

### 5.3 收敛阶

超线性收敛，收敛阶 $p = \frac{1+\sqrt{5}}{2} \approx 1.618$（黄金比例）。

| vs Newton | 优点 | 缺点 |
|-----------|------|------|
| 无需导数 | ✓ | |
| 收敛速度 | 慢一点 | 1.618 vs 2 |
| 初始值 | 需要两个 | |

---

## 六、Newton 下山法

### 6.1 问题

Newton 法单步跨度过大可能导致发散。

### 6.2 改进

引入**下山因子** $\lambda \in (0, 1]$：

$$
x_{k+1} = x_k - \lambda\frac{f(x_k)}{f'(x_k)}
$$

逐步减小 $\lambda$（如折半）直到 $|f(x_{k+1})| < |f(x_k)|$。

> 保证函数值单调下降，扩大收敛域。

---

## 七、重根的处理

设 $x^*$ 是 $f(x)$ 的 $m$ 重根：

| 方法 | 公式 | 收敛阶 |
|------|------|--------|
| 标准 Newton | $x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)}$ | 线性 |
| 修正 Newton | $x_{k+1} = x_k - m\frac{f(x_k)}{f'(x_k)}$ | 二阶 |
| 函数变形 | 对 $u(x) = \frac{f(x)}{f'(x)}$ 用 Newton | 二阶 |

---

## 八、方法总结

| 方法 | 收敛阶 | 需要导数 | 对初值敏感 | 可靠性 |
|------|--------|----------|------------|--------|
| **二分法** | 1（线性） | 否 | 否 | ★★★★★ |
| **不动点迭代** | 1（线性） | 否 | 视 $\varphi$ | ★★★ |
| **Newton 法** | 2（平方） | 是 | 是 | ★★★ |
| **割线法** | ~1.618 | 否 | 是 | ★★★ |
| **下山法** | 同 Newton | 是 | 缓解 | ★★★★ |

> 实际策略：**先用二分法缩小区间，再用 Newton 法加速收敛**。

---

## 九、小结

| 概念 | 关键点 |
|------|--------|
| 二分法 | 简单可靠，线性收敛 |
| 不动点 | $\varphi'(x^*) \ne 0$ → 线性收敛 |
| Newton 法 | 单根二阶收敛，需导数 |
| 割线法 | 差商代替导数，超线性 ~1.618 |
| 重根 | $m$ 重根 → Newton 退化为线性 |
| 下山法 | 保证 $|f|$ 递减，改善收敛域 |
