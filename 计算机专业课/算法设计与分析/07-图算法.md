# 第7章 图算法

## 7.1 图的基本概念

### 7.1.1 图的定义

**图**（Graph）G = (V, E)，其中 V 是顶点集，E 是边集。

| 类型 | 说明 |
|------|------|
| 无向图 | 边 (u, v) 无方向，(u, v) = (v, u) |
| 有向图 | 边 <u, v> 有方向，从 u 指向 v |
| 带权图 | 每条边附带一个权值 |
| 连通图 | 任意两顶点间都有路径 |
| 完全图 | 每对顶点间都有边，n 个顶点有 n(n-1)/2 条边 |

### 7.1.2 图的存储结构

#### 邻接矩阵

```c
#define N 100
int graph[N][N];  // graph[i][j] = 1(有边) 或 0(无边)
// 带权图: graph[i][j] = 权值 或 INF(无边)
```

- **优点**：O(1) 判断边是否存在
- **缺点**：O(n²) 空间，稀疏图浪费

#### 邻接表

```c
typedef struct ArcNode {
    int adjvex;           // 邻接顶点
    int weight;           // 权值
    struct ArcNode *next; // 下一条边
} ArcNode;

typedef struct {
    int data;
    ArcNode *firstarc;    // 第一条边
} VNode;

VNode adjlist[N];
```

- **优点**：O(n+e) 空间，适合稀疏图
- **缺点**：判断边存在需遍历链表

## 7.2 图的遍历

### 7.2.1 深度优先搜索（DFS）

**核心思想**：从起点出发，沿着一条路径尽可能深入，无法继续时回溯。

```c
int visited[N];

// 递归版
void DFS(int v) {
    visited[v] = 1;
    printf("%d ", v);
    
    for (int w = 0; w < n; w++) {
        if (graph[v][w] && !visited[w])
            DFS(w);
    }
}

// 非递归版（用栈）
void DFS_Stack(int start, int n) {
    int stack[N], top = -1;
    int visited[N] = {0};
    
    stack[++top] = start;
    visited[start] = 1;
    
    while (top >= 0) {
        int v = stack[top--];
        printf("%d ", v);
        
        for (int w = n - 1; w >= 0; w--) {
            if (graph[v][w] && !visited[w]) {
                stack[++top] = w;
                visited[w] = 1;
            }
        }
    }
}
```

**复杂度**：
- 邻接矩阵：O(n²)
- 邻接表：O(n + e)

### 7.2.2 广度优先搜索（BFS）

**核心思想**：从起点出发，逐层访问邻居节点（先近后远）。

```c
void BFS(int start, int n) {
    int queue[N], front = 0, rear = 0;
    int visited[N] = {0};
    
    queue[rear++] = start;
    visited[start] = 1;
    
    while (front < rear) {
        int v = queue[front++];
        printf("%d ", v);
        
        for (int w = 0; w < n; w++) {
            if (graph[v][w] && !visited[w]) {
                queue[rear++] = w;
                visited[w] = 1;
            }
        }
    }
}
```

**复杂度**：同 DFS。

### 7.2.3 DFS vs BFS

| 维度 | DFS | BFS |
|------|-----|-----|
| 数据结构 | 栈（递归） | 队列 |
| 遍历顺序 | 先深后广 | 先广后深 |
| 适用场景 | 拓扑排序、连通分量、强连通分量 | 最短路径（无权图）、层次遍历 |
| 空间 | O(h)，h为深度 | O(w)，w为最大宽度 |

## 7.3 最短路径

### 7.3.1 Dijkstra 算法

**适用**：单源最短路径，**非负权**图。

**贪心策略**：每次选择当前距离起点最近的未确定顶点，通过它松弛其他顶点。

```c
void Dijkstra(int graph[][N], int n, int start) {
    int dist[N];      // dist[i]: 起点到i的最短距离
    int visited[N];   // 是否已确定最短路径
    int path[N];      // 前驱节点（用于回溯路径）
    
    // 初始化
    for (int i = 0; i < n; i++) {
        dist[i] = graph[start][i];
        visited[i] = 0;
        path[i] = (graph[start][i] < INF) ? start : -1;
    }
    visited[start] = 1;
    dist[start] = 0;
    
    // 确定 n-1 个顶点
    for (int i = 1; i < n; i++) {
        // 找最近的未确定顶点
        int minDist = INF, u = -1;
        for (int j = 0; j < n; j++) {
            if (!visited[j] && dist[j] < minDist) {
                minDist = dist[j];
                u = j;
            }
        }
        
        if (u == -1) break;  // 不可达
        visited[u] = 1;
        
        // 松弛操作
        for (int v = 0; v < n; v++) {
            if (!visited[v] && graph[u][v] < INF &&
                dist[u] + graph[u][v] < dist[v]) {
                dist[v] = dist[u] + graph[u][v];
                path[v] = u;
            }
        }
    }
}
```

**复杂度**：
- 朴素实现：O(n²)
- 堆优化：O((n+e) log n)

### 7.3.2 Bellman-Ford 算法

**适用**：单源最短路径，**可处理负权边**（但不能有负权回路）。

核心思想：进行 n-1 轮松弛，每轮对所有边执行一次松弛操作。

```c
int BellmanFord(Edge edges[], int m, int n, int start) {
    int dist[N];
    for (int i = 0; i < n; i++) dist[i] = INF;
    dist[start] = 0;
    
    // n-1 轮松弛
    for (int i = 1; i < n; i++) {
        for (int j = 0; j < m; j++) {
            int u = edges[j].u, v = edges[j].v, w = edges[j].w;
            if (dist[u] != INF && dist[u] + w < dist[v])
                dist[v] = dist[u] + w;
        }
    }
    
    // 检查负权回路
    for (int j = 0; j < m; j++) {
        int u = edges[j].u, v = edges[j].v, w = edges[j].w;
        if (dist[u] != INF && dist[u] + w < dist[v])
            return 0;  // 存在负权回路
    }
    return 1;  // 无负权回路
}
```

**时间复杂度**：O(n·m)

### 7.3.3 Floyd-Warshall 算法

**适用**：**所有顶点对**之间的最短路径。

```c
void Floyd(int graph[][N], int n) {
    for (int k = 0; k < n; k++)          // 以k为中间点
        for (int i = 0; i < n; i++)
            for (int j = 0; j < n; j++)
                if (graph[i][k] + graph[k][j] < graph[i][j])
                    graph[i][j] = graph[i][k] + graph[k][j];
}
```

**时间复杂度**：O(n³)，但代码极其简洁。

### 7.3.4 算法对比

| 算法 | 适用范围 | 时间复杂度 | 空间复杂度 |
|------|----------|-----------|-----------|
| Dijkstra | 非负权，单源 | O(n²) / O((n+e)log n) | O(n) |
| Bellman-Ford | 可负权，单源 | O(n·m) | O(n) |
| Floyd | 任意权，全源 | O(n³) | O(n²) |

## 7.4 拓扑排序

### 7.4.1 问题描述

对有向无环图（DAG），将所有顶点排成一个线性序列，使得对于每条有向边 <u, v>，u 在序列中出现在 v 之前。

### 7.4.2 算法（基于入度）

```c
void TopologicalSort(int n) {
    int indegree[N] = {0};
    int queue[N], front = 0, rear = 0;
    int topo[N], cnt = 0;
    
    // 计算入度
    for (int i = 0; i < n; i++)
        for (ArcNode *p = adjlist[i].firstarc; p; p = p->next)
            indegree[p->adjvex]++;
    
    // 入度为0的顶点入队
    for (int i = 0; i < n; i++)
        if (indegree[i] == 0)
            queue[rear++] = i;
    
    while (front < rear) {
        int v = queue[front++];
        topo[cnt++] = v;
        
        for (ArcNode *p = adjlist[v].firstarc; p; p = p->next) {
            int w = p->adjvex;
            indegree[w]--;
            if (indegree[w] == 0)
                queue[rear++] = w;
        }
    }
    
    if (cnt < n)
        printf("图中存在回路！\n");
}
```

**时间复杂度**：O(n + e)

## 7.5 关键路径（AOE网）

### 7.5.1 基本概念

**AOE网**（Activity On Edge）：边表示活动，顶点表示事件。用于估算工程的最短完成时间。

- **ve[i]**：事件 i 的**最早发生时间**
- **vl[i]**：事件 i 的**最迟发生时间**
- **e[k]**：活动 k 的**最早开始时间**（= ve[起点]）
- **l[k]**：活动 k 的**最迟开始时间**（= vl[终点] - 持续时间）
- **关键活动**：e[k] = l[k] 的活动（不能延误）

### 7.5.2 算法

```c
void CriticalPath(int n) {
    int ve[N] = {0}, vl[N];
    int topo[N], cnt;
    
    // 1. 拓扑排序，计算 ve
    TopologicalSort(n, topo, &cnt);
    
    // 正推 ve
    for (int i = 0; i < cnt; i++) {
        int u = topo[i];
        for (ArcNode *p = adjlist[u].firstarc; p; p = p->next) {
            int v = p->adjvex;
            if (ve[u] + p->weight > ve[v])
                ve[v] = ve[u] + p->weight;
        }
    }
    
    // 2. 反推 vl
    for (int i = 0; i < n; i++) vl[i] = ve[n-1];
    for (int i = cnt - 1; i >= 0; i--) {
        int u = topo[i];
        for (ArcNode *p = adjlist[u].firstarc; p; p = p->next) {
            int v = p->adjvex;
            if (vl[v] - p->weight < vl[u])
                vl[u] = vl[v] - p->weight;
        }
    }
    
    // 3. 输出关键活动
    for (int u = 0; u < n; u++) {
        for (ArcNode *p = adjlist[u].firstarc; p; p = p->next) {
            int v = p->adjvex;
            int e = ve[u];
            int l = vl[v] - p->weight;
            if (e == l)
                printf("关键活动: %d→%d\n", u, v);
        }
    }
}
```

## 7.6 最大流（Ford-Fulkerson）

### 7.6.1 问题描述

在流网络中，从源点 s 到汇点 t，每条边有容量限制，求最大流量。

### 7.6.2 增广路算法

```c
int FordFulkerson(int cap[][N], int n, int s, int t) {
    int flow[N][N] = {0};  // 当前流
    int maxFlow = 0;
    
    while (1) {
        int parent[N];  // BFS找增广路
        int visited[N] = {0};
        int queue[N], front = 0, rear = 0;
        
        queue[rear++] = s;
        visited[s] = 1;
        parent[s] = -1;
        
        while (front < rear) {
            int u = queue[front++];
            for (int v = 0; v < n; v++) {
                if (!visited[v] && cap[u][v] - flow[u][v] > 0) {
                    parent[v] = u;
                    visited[v] = 1;
                    queue[rear++] = v;
                    if (v == t) break;  // 找到增广路
                }
            }
        }
        
        if (!visited[t]) break;  // 无增广路
        
        // 找瓶颈容量
        int bottleneck = INF;
        for (int v = t; v != s; v = parent[v]) {
            int u = parent[v];
            bottleneck = min(bottleneck, cap[u][v] - flow[u][v]);
        }
        
        // 更新流
        for (int v = t; v != s; v = parent[v]) {
            int u = parent[v];
            flow[u][v] += bottleneck;
            flow[v][u] -= bottleneck;  // 反向边
        }
        
        maxFlow += bottleneck;
    }
    return maxFlow;
}
```

### 7.6.3 最大流最小割定理

> 网络中的**最大流量**等于**最小割的容量**。

> 对下图用 Dijkstra 算法求顶点 0 到其他各顶点的最短路径。
>
> ```
>     0 --10-- 1 --50-- 4
>     | \      |       /
>    30 100   20    10
>     |    \   |   /

| 3 | {0,1,2,3} | 0 | 10 | 30 | 30 | 40 |

- 0→1: 10

### 习题3

> 给出求无向图连通分量数的算法。

**解**：

```c
int CountComponents(int n) {
    int visited[N] = {0};
    int count = 0;
    
    for (int i = 0; i < n; i++) {
        if (!visited[i]) {
            DFS(i, visited);  // 或 BFS
            count++;
        }
    }
    return count;
}
```

对每个未访问顶点启动一次 DFS/BFS，每次遍历一个完整连通分量。

**时间复杂度**：O(n + e)
