## 1.1 算法的概念

### 1.1.1 什么是算法

**算法**（Algorithm）是指解题方案的准确而完整的描述，是一系列解决问题的清晰指令。算法代表着用系统的方法描述解决问题的策略机制。

一个算法必须具备以下五个重要特性：

| 特性 | 说明 |
|------|------|
| **有穷性**（Finiteness） | 算法必须在执行有限步之后终止，不能无限循环 |
| **确定性**（Definiteness） | 算法的每一步都必须有确切的定义，不存在歧义 |
| **可行性**（Effectiveness） | 算法中的每一条指令都必须是能够精确执行的 |
| **输入**（Input） | 一个算法有0个或多个输入 |
| **输出**（Output） | 一个算法有1个或多个输出，反映输入数据的加工结果 |

### 1.1.2 算法与程序的区别

- **算法**：是一组逻辑步骤的抽象描述，独立于任何编程语言
- **程序**：是算法在某种特定编程语言下的具体实现

算法是程序的灵魂，程序是算法的载体。

## 1.2 算法复杂性分析

### 1.2.1 时间复杂度

**时间复杂度**是指执行算法所需要的计算工作量，通常用输入规模 n 的函数 T(n) 来表示。

常见的分析方法：

#### （1）事后统计法

运行程序并测量实际执行时间。**缺点**：依赖硬件环境、编程语言、编译器优化等因素，不具有普适性。

#### （2）事前分析法

对算法语句的**执行频度**进行理论分析，用 O、Ω、Θ 等渐近符号表示。

### 1.2.2 渐近符号

#### 大 O 符号（上界）

> **定义**：T(n) = O(f(n)) 当且仅当存在正常数 c 和 n₀，使得对于所有 n ≥ n₀，有 T(n) ≤ c·f(n)。

大 O 表示算法的**最坏情况**时间复杂度上界。

#### 大 Ω 符号（下界）

> **定义**：T(n) = Ω(f(n)) 当且仅当存在正常数 c 和 n₀，使得对于所有 n ≥ n₀，有 T(n) ≥ c·f(n)。

大 Ω 表示算法的**最好情况**下界。

#### Θ 符号（紧确界）

> **定义**：T(n) = Θ(f(n)) 当且仅当 T(n) = O(f(n)) 且 T(n) = Ω(f(n))。

Θ 表示算法复杂度的**精确阶**。

### 1.2.3 常见时间复杂度排序（从优到劣）

```
O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(n³) < O(2ⁿ) < O(n!) < O(nⁿ)
```

| 复杂度 | 名称 | 典型算法 |
|--------|------|----------|
| O(1) | 常数阶 | 哈希查找 |
| O(log n) | 对数阶 | 二分查找 |
| O(n) | 线性阶 | 顺序查找 |
| O(n log n) | 线性对数阶 | 归并排序、快速排序（平均） |
| O(n²) | 平方阶 | 冒泡排序、插入排序 |
| O(n³) | 立方阶 | Floyd算法、普通矩阵乘法 |
| O(2ⁿ) | 指数阶 | 穷举搜索 |
| O(n!) | 阶乘阶 | 旅行商问题暴力求解 |

---

### 1.2.4 空间复杂度

**空间复杂度** S(n) 是指算法执行过程中所需的存储空间，包括：

- **固定部分**：指令空间（代码）、简单变量、常量
- **可变部分**：输入数据占用空间、递归栈空间、动态分配的辅助空间

## 1.3 递归与递归方程

### 1.3.1 递归的概念

**递归**是指一个过程或函数在其定义中直接或间接调用自身的一种方法。

递归必须满足两个基本条件：
1. **递归边界**（终止条件）—— 防止无穷递归
2. **递归关系**（递推公式）—— 将大问题转化为小问题

**经典示例 — 阶乘函数**：

```
n! = 1              (当 n = 0)
n! = n × (n-1)!    (当 n > 0)
```

**经典示例 — Fibonacci数列**：

```
F(0) = 0
F(1) = 1
F(n) = F(n-1) + F(n-2)   (n ≥ 2)
```

### 1.3.2 递归方程求解

#### 方法一：迭代展开法

对递归关系反复代入，直到到达边界条件。

> **例题**：求解 T(n) = T(n-1) + n，T(1) = 1

**解**：
```
T(n) = T(n-1) + n
     = T(n-2) + (n-1) + n
     = T(n-3) + (n-2) + (n-1) + n
     = ...
     = T(1) + 2 + 3 + ... + n
     = 1 + 2 + 3 + ... + n
     = n(n+1)/2
     = Θ(n²)
```

#### 方法二：Master定理（主定理）

对于形如 **T(n) = a·T(n/b) + f(n)** 的递归式：

| 条件 | 结论 |
|------|------|
| 若 f(n) = O(n^(log_b a - ε))，ε > 0 | T(n) = Θ(n^(log_b a)) |
| 若 f(n) = Θ(n^(log_b a)) | T(n) = Θ(n^(log_b a) · log n) |
| 若 f(n) = Ω(n^(log_b a + ε))，ε > 0，且 af(n/b) ≤ cf(n)，c < 1 | T(n) = Θ(f(n)) |

> **例题**：求解 T(n) = 2T(n/2) + n

**解**：a = 2, b = 2, f(n) = n

n^(log_b a) = n^(log₂ 2) = n¹ = n

f(n) = n = Θ(n)，属于情况2

∴ **T(n) = Θ(n log n)**

这正好是归并排序的时间复杂度。

> **例题**：求解 T(n) = 9T(n/3) + n

**解**：a = 9, b = 3, f(n) = n

n^(log_b a) = n^(log₃ 9) = n²

f(n) = n = O(n^(2-1)) = O(n¹)，属于情况1

∴ **T(n) = Θ(n²)**

> **例题**：求解 T(n) = T(2n/3) + 1

**解**：a = 1, b = 3/2, f(n) = 1

n^(log_b a) = n^(log_{3/2} 1) = n⁰ = 1

f(n) = 1 = Θ(1)，属于情况2

∴ **T(n) = Θ(log n)**

这正是二分查找的时间复杂度。

#### 方法三：递归树法

将递归过程画成树形结构，每层求和，最后累加所有层次。

> **例题**：用递归树法求解 T(n) = 3T(n/4) + cn²

**解**：

```
             cn²                    ← 第0层: cn²
     ┌───────┼───────┐
  T(n/4)    T(n/4)    T(n/4)        ← 第1层: 3·c(n/4)² = (3/16)cn²
  ┌──┼──┐   ┌──┼──┐   ┌──┼──┐
T(n/16)... ...       ...           ← 第2层: 9·c(n/16)² = (9/256)cn²
   ...
```

总代价 = cn² · [1 + 3/16 + (3/16)² + ...]（无穷等比级数）
       = cn² · 1/(1 - 3/16)
       = (16/13)cn²
       = **Θ(n²)**

## 1.4 算法设计的基本策略

| 策略 | 核心思想 |
|------|----------|
| **分治法** | 将大问题分解为若干个小问题，分别求解后合并 |
| **动态规划** | 将问题分解为重叠子问题，自底向上求解并存储中间结果 |
| **贪心算法** | 每一步都选择当前看起来最优的解 |
| **回溯法** | 深度优先搜索解空间树，遇到非可行解时回溯 |
| **分支限界法** | 广度优先或最小代价优先搜索解空间树 |

## 1.5 本章习题

### 习题1

> 分析下面程序段的时间复杂度。

```c
for (i = 1; i <= n; i++)
    for (j = 1; j <= i; j++)
        x = x + 1;
```

**解**：

内层循环执行次数 = 1 + 2 + 3 + ... + n = n(n+1)/2

∴ 时间复杂度为 **O(n²)**。

### 习题2

> 求解递归方程：T(n) = 4T(n/2) + n²

**解**：a = 4, b = 2, f(n) = n²

n^(log_b a) = n^(log₂ 4) = n²

f(n) = n² = Θ(n²)，Master定理情况2

∴ **T(n) = Θ(n² log n)**

### 习题3

> 求解递归方程：T(n) = T(n/2) + n

**解**：a = 1, b = 2, f(n) = n

n^(log_b a) = n^(log₂ 1) = n⁰ = 1

f(n) = n = Ω(n^(0+1)) = Ω(n)，情况3

验证正则条件：af(n/b) = 1·n/2 = n/2 ≤ cn（取 c=1/2 < 1），满足。

∴ **T(n) = Θ(n)**
